# **App Name**: Feedz

## Core Features:

- Ingredient Display: Showcase all available feed ingredients with detailed descriptions.
- Stock Management: Admin panel for stock level management, showing 'in-stock' and 'out-of-stock' status.
- Combination Recommendations: AI powered tool to provide suggested combinations of ingredients based on animal type and nutritional goals.
- SEO Optimized Design: Clean, modern, and SEO-friendly design for optimal user experience and search engine visibility.

## Style Guidelines:

- Primary color: Muted olive green (#A3B18A) for a natural and trustworthy feel.
- Background color: Light beige (#F5F5DC) to provide a neutral and clean backdrop.
- Accent color: Burnt orange (#D18935) to highlight important interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif font providing a clean and modern feel.
- Use clear and simple icons to represent different ingredient categories and actions.
- Responsive layout to ensure optimal viewing experience on all devices.
- Subtle transitions and animations for a smooth user experience.
